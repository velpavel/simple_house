Работает автоматически по движению.
Для настройки цвета, яркости, ручного включения и выключения используется
Bluetooth Electronics
https://play.google.com/store/apps/details?id=com.keuwl.arduinobluetooth&hl=ru

Известная проблема - нет ОС на элементы управленяи пульта. При подключении пульт показывает дефолтные значения, а не текущие.

Скрин моего вида пульта прилагается.
